# microservices-auth
microservices authentication system with graphql gateway, mailer service and nextjs client using rabbitMQ for messaging
## tech stack

- node.js
- typescript
- postgreSQL
- redis
- rabbitMQ (amqp)
- apollo graphql
- express
- jwt
- type-graphql
- typeorm
- nodemailer
- nextjs
- react
- mui
- formik
